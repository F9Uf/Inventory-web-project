# Inventory Project

Project for CPE231 Database System

## Requirement
 - **Node.js:**  ^10.15.3
 - **Npm:** ^6.9.0

## How to run
### Client
  ```
  $ cd client
  $ npm install
  $ npm run serve
  ```
### Server
  ```
  $ cd server
  $ npm install
  $ npm run dev (for dev with nodemon)
  $ npm start (for start)
